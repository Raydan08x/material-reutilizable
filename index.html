      Aprende algo de Bootstrap

      Que pone Bootstrap y que debes entender tu

      Bootstrap acelera clases visuales, pero no reemplaza conceptos. Este bloque te ayuda a reconocer la estructura que Bootstrap te da y la logica que igual debes dominar con HTML, CSS y JS.

      .container / .row / .col

      Te ensena composicion y grid. Luego puedes replicarlo con display: grid o flex.

      .card / .badge / .btn

      Sirven para entender jerarquia, espaciado, variantes visuales y reutilizacion de componentes.

      .modal / .accordion / .toast

      Son una puerta para practicar estados, capas, foco, eventos y comportamiento interactivo real.

      • Primero mira la version resuelta.
      • Luego identifica que parte es solo estructura HTML.
      • Despues separa que hace Bootstrap y que hace tu CSS.
      • Al final reescribe el comportamiento en JavaScript sin depender de magia externa.
      Laboratorio Bootstrap

      Traduce clases de Bootstrap a decisiones reales de maquetacion

      Aqui no solo ves nombres de clases: ves que problema resuelven y con que herramientas nativas puedes reconstruir el mismo patron.

      Layout .container .row .col-md-6

      Equivale a pensar en ancho maximo, separacion interna, columnas y breakpoint.

      HTML semantico + display: grid + @media
      Card .card .card-body .shadow

      Equivale a una superficie con jerarquia, padding, radio, borde y profundidad visual.

      background + padding + border-radius + box-shadow
      Acciones .btn .btn-primary .badge

      Equivale a estados visuales, contraste, feedback hover y pequenos indicadores.

      Variables CSS + estados :hover + chips reutilizables

      Actividad 1

      Conversor C, F y K

      PDF actividad

      Actividad guiada de lógica en JavaScript: recibir grados Celsius, validar que el dato sea numérico y mostrar Fahrenheit y Kelvin en pantalla. Se puede resolver con prompt o con input; aquí quedó implementado con interfaz visual.

      Selecciona una unidad y calcula.
      Abre la funcionalidad 44 con el paso a paso completo.

      Paso a paso sugerido

      1. Pide al usuario la temperatura en Celsius con un input o con prompt.
      2. Convierte el valor con Number(...) y valida con Number.isFinite.
      3. Si el dato no es numérico, muestra un error y vuelve a solicitar el valor.
      4. Si es válido, calcula Kelvin con c + 273.15 y Fahrenheit con (c * 9 / 5) + 32.
      5. Imprime ambos resultados en el DOM o en consola y prueba casos como 45 y 14.
