      Guia de implementacion

      Que cambiar para reutilizar este proyecto

      1. Cambios principales en HTML

      Cambia textos, ids y atributos data-section. Si agregas una nueva seccion, crea un boton con el mismo valor en data-section.

      <button data-section="servicios">Servicios</button>
<section id="servicios" data-view-section>...</section>

      2. Cambios principales en CSS

      Modifica las variables dentro de :root para cambiar colores, radios, sombras y tamaño de navbar.

      :root {
  --primary: #2563eb;
  --secondary: #7c3aed;
  --radius-lg: 24px;
}

      3. Cambios principales en JS modular

      Agrega clases en funciones.js usando export y llamalas desde script.js con import.

      import { CustomSelect } from "./funciones.js";
new CustomSelect("[data-custom-select]");

      4. Version normal sin modulos

      Usa script-normal.js quitando type="module" si tu profesor no quiere modulos.

      <script src="script-normal.js"></script>
